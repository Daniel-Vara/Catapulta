📖 O que é uma Função do 2º Grau?

A função do segundo grau tem a forma h(t) = at² + bt + c, onde a ≠ 0. Seu gráfico é uma parábola.

  • Se a > 0 → concavidade para cima 📈
  • Se a < 0 → concavidade para baixo 📉

🚀 No movimento da catapulta, a altura da pedra segue essa lei, pois a aceleração da gravidade é constante.

📘 Nossa Trajetória

A nossa catapulta foi montada utilizando palitinhos de madeira, uma tampa de garrafa (guaraná), um elástico e cola quente.

Após a montagem, realizamos o lançamento de uma bolinha de cola quente. Observamos que após 1 segundo, ela atingiu 12 cm de altura, iniciando sua descida.

Modelamos o fenômeno com a função: h(t) = at² + bt (pois parte do chão, então c = 0)

  • h(1) = 12 cm
  • t = 1 é o vértice (ponto mais alto)

tv = -b / (2a) = 1 → -b = 2a → b = -2a

h(1) = a(1)² + b(1) = a + b = 12

Substituindo b: a + (-2a) = 12 → -a = 12 → a = -12

b = -2(-12) = 24

Função final: h(t) = -12t² + 24t

💡 Dica: O tempo total de voo é quando h(t) = 0 → -12t² + 24t = 0 → t(24 - 12t) = 0 → t = 0 ou t = 2 segundos
