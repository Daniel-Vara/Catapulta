📖 O que é uma Função do 2º Grau?

A função do segundo grau tem a forma h(t) = at² + bt + c, onde a ≠ 0. Seu gráfico é uma parábola.

  • Se a > 0 → concavidade para cima 📈
  • Se a < 0 → concavidade para baixo 📉

🚀 No movimento da catapulta, a altura da pedra segue essa lei, pois a aceleração da gravidade é constante.
